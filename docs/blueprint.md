# **App Name**: Masterpiece AI

## Core Features:

- Dual AI Image Generation: Generate two AI images (one from Gemini and one from OpenAI) based on user text prompt, aspect ratio, and size. Store the prompt used to generate images to show in history section.
- Image Management: Enable users to download, share, or regenerate created images, with an option to store generation history tied to the user account using Supabase.
- AI Image Enhancement: Feature a tool for image quality enhancement up to 16K, offering a before-and-after comparison to showcase improvements, powered by Gemini and Open AI.
- Background Removal Tool: Integrate a background removal tool, utilizing available APIs or AI tool that can remove backgrounds effectively for 24/7 operation.
- AI Product Photoshoot: AI Product Photoshoot: Accepts uploaded product image and use it to generate professional-looking photoshoot images in a different environment or scene, leveraging both Gemini and Open AI. Produce two photos for each prompt.
- AI Interior Design: AI Interior Designer: Generate redesign options for uploaded images of interior rooms based on user prompts for style and aesthetics, leveraging both Gemini and Open AI. Produce two photos for each prompt.
- AI Avatar Generation: AI Avatar Generator: Transform uploaded personal photos into stylized avatars with themes specified by the user, with both Gemini and Open AI providing output. Generate two avatars for each prompt
- User Authentication System: Implement a secure login system integrated with Supabase for user authentication, registration, and account management.

## Style Guidelines:

- Primary color: HSL(195, 70%, 50%) - RGB(30, 175, 209), a vibrant cyan to evoke creativity and technological advancement.
- Background color: HSL(195, 20%, 95%) - RGB(240, 247, 249), a light tint of cyan to create a calm and spacious feel.
- Accent color: HSL(225, 50%, 50%) - RGB(92, 115, 191), a cool blue-violet to complement the primary color and add a touch of sophistication.
- Font pairing: 'Space Grotesk' (sans-serif) for headers and 'Inter' (sans-serif) for body text to offer a clean, modern, and highly readable interface.
- Use a set of consistent, minimalist icons that are easily recognizable and align with the application's modern aesthetic. These should have a thin stroke weight and a geometric style to complement the chosen typography.
- Implement a responsive design that adapts smoothly to various screen sizes. The layout should emphasize ease of navigation, with a clean, intuitive interface that prioritizes user content. Employ a grid-based system to maintain consistency across all pages and sections.
- Incorporate subtle transitions and animations to enhance user engagement without distracting from the core functionality. This could include gentle fades, slides, or scaling effects for button hovers, loading indicators, and content updates.
